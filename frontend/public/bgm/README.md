# BGM Files

이 폴더에 배경음악 파일(.ogg)을 추가하세요.

## 필요한 파일
- `lobby1.ogg` ~ `lobby99.ogg` - 로비 배경음악 (랜덤 재생)

## 파일 형식
- 오직 .ogg 파일만 지원합니다
- 파일명은 `lobby1.ogg`, `lobby2.ogg`, ... `lobby99.ogg` 형식으로 작성해주세요

## 사용 방법
1. .ogg 형식의 음악 파일을 이 폴더에 추가
2. 파일명을 `lobby1.ogg`, `lobby2.ogg`, `lobby3.ogg` 등으로 설정
3. 애플리케이션을 새로고침하면 랜덤으로 선택된 BGM이 자동 재생됩니다

## 예시
```
/bgm/lobby1.ogg
/bgm/lobby2.ogg
/bgm/lobby3.ogg
...
/bgm/lobby99.ogg
```

원하는 만큼만 추가하면 됩니다 (1개부터 99개까지)
